# خطة جعل القائمة متفاعلة ومتجاوبة

## الهدف
جعل NavMenu متجاوب مع جميع الشاشات (هواتف، أجهزة لوحية، سطح المكتب)

## التغييرات المطلوبة

### 1. NavMenu.razor
- إضافة المنطق التفاعلي (Blazor `ToggleMenu` method)
- تحويل `onclick="this.parentElement.classList.toggle('active')"` إلى `@onclick="() => ToggleMenu()"`
- إضافة `openMenus` HashSet لتتبع القوائم المفتوحة
- إضافة زر إغلاق (X) للقائمة في upper corner
- إضافة `NavMenuCssClass` للتحكم بإخفاء/إظهار على الهواتف

### 2. MainLayout.razor
- إضافة زر همبرغر (menu) في top-row
- إضافة overlay للتغطية عند فتح القائمة على الهواتف
- إضافةmethods: `ToggleSidebar`, `CloseSidebarOnMobile`

### 3. NavMenu.razor.css
- إضافةMedia Query للشاشات الصغيرة:
  - `aside`: إخفاء默认 وإظهار عند الطلب
  - `transform: translateX(100%)` للإخفاء
  - `transform: translateX(0)` للإظهار
- إضافة أنماط `.menu-content` للقوائم المنسدلة
- أنماط Overlay

### 4. MainLayout.razor.css
- إضافة أنماطOverlay
- دعم الشاشات الصغيرة

## الكود المطلوب

### NavMenu.razor - @code
```csharp
@code {
    private bool collapseNavMenu = false;
    private HashSet<string> openMenus = new();

    private string? NavMenuCssClass => collapseNavMenu ? "translate-x-full md:translate-x-0" : "translate-x-0";

    private void ToggleNavMenu() {
        collapseNavMenu = !collapseNavMenu;
    }

    private void ToggleMenu(string menuName) {
        if (openMenus.Contains(menuName))
            openMenus.Remove(menuName);
        else
            openMenus.Add(menuName);
    }
}
```
